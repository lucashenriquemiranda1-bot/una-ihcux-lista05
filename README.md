Este projeto aplica conceitos de IHC e UX  em um sistema simples de conversão de moeda no console

O sistema mantém o usuário informado sobre o que está acontecendo com mensagens como 
[SISTEMA]: Conectando ao Banco Central...
[SISTEMA]: Calculando taxas...

O sistema trata entradas inválidas para evitar falhas com o uso de try-catch 

E esteticamente falando  a Interface é  simples, limpa e focada no essencial.
